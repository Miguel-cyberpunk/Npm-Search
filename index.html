¿Qué es una Etiqueta de HTML?

Son fragmentos de código que permiten crear elementos HTML, que son estructura básicas de este lenguaje. Estos elementos tienen dos propiedades básicas: los "Atributos" y el "Contenido". Para que un elemento sea válido, normalmento tiene una etiqueta de inicio "< nombre-de-elemento >"y una etiqueta de cierre "< /nombre-de-elemento >"

Etiqueta "Body"

El elemento "body" representa el contenido de un documento HTML. Solo puede haber un elemento "body" en un documento.

Un Consejo con Respecto al Uso de las Minúsculas

Si bien el HTML acepta tanto etiquetas en mayúsculas "B" como en minúsculas "b" el Wold Wide Web Consorotium (W3C) recomienda el uso de las minúsculas. Es por ello que se recomienda acostumbrarse a usar minúsculas cuando escribamos códigos en HTML.
